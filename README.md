## 🧠 Was macht das Projekt?

Das Projekt soll zeigen, wie ein einfaches neuronales Netz aus Beispieldaten lernen kann, um Schachstellungen besser einzuschätzen.  
Dabei liegt der Fokus auf dem **Lernprozess**, nicht auf einer perfekten Schach-KI.
